Some of my works

2020

Simple marketplace web application prototype

This web application was built using NodeJS + MongoDB for the backend side and ReactJS for the frontend site. The features of this web application is user can see all available products and filter it by its category, then user can add, customize the quantity and remove multiple items in the cart, and user can do checkout from their cart and see their order histories. And yes, this web application is already using a responsive front end, so it's can also used using mobile.
